# Словарь AIST POS — единый глоссарий

Единый источник терминов для всех текстов репозитория. Два слоя:

- **Пользовательский слой** — всё, что видит пользователь: вопросы и уровни рубрики (`ask`, `levels`, `title`, `actions.to_L3`), выдача оценки (Фаза 4 assess), `description` навыков, интервью setup, шаблоны `setup-templates/`, README, шаблоны вывода (бриф, задачи).
- **Внутренний слой** — инструкции SKILL.md агенту, config, схемы (`entities-schema.md`), `connectors.md`, комментарии YAML.

## Правила

1. В пользовательском слое — только термины из колонки «Пользователю». Слова из «Запрещено» не печатать и не писать в файлы пользователя.
2. Во внутреннем слое допустимы термины из колонки «Internal» — но всегда одни и те же (не синонимить).
3. Новый термин → сначала строка в этом файле, потом использование.
4. Целевая аудитория пользовательского слоя — руководитель без техфона: незнакомы не только термины, но и сами подходы. Формулируй через узнаваемый бытовой исход, «зачем» встраивай прямо в текст.
