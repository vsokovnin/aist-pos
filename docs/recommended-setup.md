# Рекомендованная настройка — коннекторы и плагины

> Подключаются вручную в Claude (OAuth / установка плагина — интерактивный шаг, навык сделать это за вас не может). Полный актуальный каталог — **claude.com/plugins** и Connector directory в самом приложении. Ниже — ориентир «must-have» для руководителя.

## Базовые коннекторы (baseline — подключите первыми)
- **Google Workspace** (Gmail, Calendar, Drive, Docs) **или** **Microsoft 365** (Outlook, OneDrive, SharePoint, Teams) — почта + календарь + файлы. Без них не работают утренний бриф, подготовка к встрече и бронь календаря.
- *(опционально)* **Telegram** — если используете его как рабочий канал.

## Must-have плагины (быстрый понятный value)
1. **Productivity** — задачи и календарь в дневной фокус. Прямой fit для нескольких ролей/советов сразу.
2. **Enterprise search** — поиск ответа по всем докам и инструментам без ручного перелопачивания.
3. **Finance** (или **Financial analysis**) — чтение P&L, финмодели, сравнительный анализ. Для члена СД — разбор материалов к заседанию (выводы проверяйте с профессионалом).
4. *(опционально)* **DocuSign** (коннектор) — отправка и трекинг подписей: договоры, согласия.
5. *(опционально)* **Sales** — ресёрч по контактам и подготовка к переговорам.

**Топ-3 «за один вечер»:** Google Workspace / M365 → Productivity → Enterprise search.

*Источник: официальный каталог плагинов Anthropic (claude.com/docs/plugins) и Claude connector directory; каталог быстро растёт — сверяйтесь с актуальным списком в приложении.*
