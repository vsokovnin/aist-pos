# Рекомендованная настройка — коннекторы и плагины

> Подключаются вручную в Claude (OAuth / установка плагина — интерактивный шаг, навык сделать это за вас не может). Полный актуальный каталог — **claude.com/plugins** и Connector directory в самом приложении. Ниже — ориентир «must-have» для индивидуального руководителя.

## Базовые коннекторы (baseline — подключите первыми)
- **Google Workspace** (Gmail, Calendar, Drive, Docs) — **рекомендованный дефолт для индивидуального пользователя**: доступен на всех планах по личному OAuth и единственный с реальной записью (создание событий календаря, черновики писем). Без почты и календаря не работают утренний бриф, подготовка к встрече и бронь календаря.
- **Microsoft 365** (Outlook, OneDrive, SharePoint, Teams) — **только если вы внутри корпоративного тенанта**: требует плана Team/Enterprise и прав Global Admin в Microsoft Entra, и работает **только на чтение** (создавать/отправлять нельзя). Для приватности — local connector (трафик не идёт через серверы Anthropic).
- *(опционально)* **Telegram** — если используете его как рабочий канал.

> Бронь календаря (сценарий `aist-pos-schedule`) реалистична на Google (есть запись); на M365 календарь только для чтения — события придётся ставить вручную.

## Power-ups (опциональные нативные плагины поверх AIST POS)
Опенсорсные плагины Anthropic, доступные на платном личном плане, дают быстрый value:
- **Finance** (`financial-analysis`) — самый additive под члена СД: аудит финмодели (`/debug-model`), `/comps`, `/dcf`, разбор board-pack. Работает на загруженных файлах без платных data-коннекторов; для публичных эмитентов США — бесплатный EdgarTools (SEC).
- **Enterprise search** (плагин) — federated-поиск `/search` и сводка `/digest` по подключённым источникам (без отдельного индекса). *(Управляемая «Ask Your Org» — только Team/Enterprise.)*
- **Productivity** — задачи + двухуровневая память + локальный дашборд. Пересекается с базой AIST POS; берите, если не выстраиваете собственный слой задач.

**Топ-3 «за один вечер»:** Google Workspace → Finance (если член СД) → Enterprise search.

*Источник: каталог плагинов Anthropic (claude.com/docs/plugins) и Claude connector directory; каталог быстро растёт — сверяйтесь с актуальным списком в приложении.*
