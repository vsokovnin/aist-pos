# AIST POS — подготовка к встрече

Читает `config/aist-pos.config.yaml`. Коннекторы — через `config/connectors.md`.

## Когда запускать
- «подготовь к встрече с X», перед внешней встречей (по календарю на сегодня).

## Нативная альтернатива
Плагин **Finance** даёт `meeting-prep-agent` (с финансовым уклоном). Этот навык — общий: готовит на вашем графе сущностей и переписке, без привязки к финансовым данным.

## Поток
1. **Цель встречи.** Из календаря (`connectors.calendar`) ближайшая/указанная встреча: кто, когда, тема. Или X задан пользователем.
2. **Кто это (база контактов).** Подними карточки участников и их организацию/проекты из `stores.entities` (через `aist-pos-entities`). Неизвестных пометь.
3. **Контекст.** Последние касания: письма (`connectors.email`) и транскрипты с этими людьми за разумное окно; открытые обязательства из `stores.tasks` с ними.
4. **Бриф к встрече:**
   ```
   🤝 Встреча: <тема> — <дата/время>
   👤 Кто: <имя — роль — организация; ключевое из карточки>
   🧵 Контекст: <2-4 пункта последних касаний>
   ⏳ Открытые обязательства: <мои/их по этим людям>
   📋 Повестка (черновик): <3-5 пунктов>
   ```
5. **Выход** — в ответ (опционально на `output_surface`).

## Правила
- Только чтение; ничего не записывает (граф обновляет US5 отдельно).
- Нет данных по человеку -> честно «нет в базе контактов», предложи US5; не выдумывай факты о людях.
