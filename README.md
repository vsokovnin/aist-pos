# AIST POS

**Личная операционная система для руководителя — поверх Claude.**

Оцените уровень ИИ-зрелости ваших рабочих процессов и выстраивайте их развитие по шагам — от точечного использования чат-бота до системы, которая помнит ваш контекст, разбирает входящие в задачи, готовит вас к встречам и держит под рукой ваши контакты и компании.

## Что это
Набор портативных навыков (skills) для **Claude Cowork / Claude Code**:
- **оценка ИИ-зрелости процессов** (`aist-pos-assess`) — ваш профиль, текущий уровень и индивидуальный план развития;
- **рабочие сценарии:** разбор переписки в задачи и обязательства, утренний бриф, бронь календаря, подготовка к встрече, база контактов и компаний.

Не SaaS и не облако: всё работает локально, на ваших файлах и подключённых сервисах. Приватность и персонализация — by design.

## Как это работает
1. Склонируйте репозиторий (`git clone`).
2. Запустите `aist-pos-setup` — короткое интервью и установка базы (корневой файл, конвенции, git, хранилища данных).
3. Запустите `aist-pos-assess` — оцените текущий уровень ИИ-зрелости и получите индивидуальный план развития.
4. Используйте рабочие сценарии по этому плану.

## Установка
- **Claude Cowork:** подключите папки из `skills/` как навыки; скопируйте `config/aist-pos.config.example.yaml` → `aist-pos.config.yaml` (или доверьте это `aist-pos-setup`); подключите нужные сервисы по `config/connectors.md`.
- **Claude Code:** положите `skills/*` в директорию навыков; конфигурация — `config/aist-pos.config.yaml`.

## Навыки
| Навык | Что делает |
|---|---|
| aist-pos-setup | Установка: интервью + база (9 базовых способностей) |
| aist-pos-assess | Оценка ИИ-зрелости процессов → профиль, уровень и план развития |
| aist-pos-entities | База контактов: организация (основа) ← люди, проекты |
| aist-pos-extract-tasks | Переписка и транскрипты → задачи и обязательства |
| aist-pos-brief | Утренний бриф (календарь + почта + обязательства) |
| aist-pos-schedule | Бронь календаря (с подтверждением) |
| aist-pos-meeting-prep | Подготовка к встрече (кто / контекст / повестка) |

## Модель ИИ-зрелости
9 базовых способностей, 3 направления, 5 стадий развития — `docs/capability-model.md`. Единый словарь терминов — `docs/glossary.md`.

Нужна **полная матрица на 40 параметров** (9 групп, продвинутый self-assessment)? — [aist-mm](https://github.com/vsokovnin/aist-mm). Здесь, в aist-pos, — упрощённый онбординг на 9 ключевых способностях с готовым диалоговым движком оценки.
