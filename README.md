# AIST POS

**Личная операционная система для руководителя — поверх Claude.**

Оцените уровень ИИ-зрелости ваших личных рабочих процессов и выстраивайте их развитие по шагам — от точечного использования чат-бота до системы, которая помнит ваш контекст, разбирает входящие в задачи, готовит вас к встречам и ведёт граф ваших людей и компаний.

## Что это
Набор портативных навыков (skills) для **Claude Cowork / Claude Code**:
- **оценка зрелости** (`aist-pos-assess`) — ваш профиль, текущий уровень и индивидуальный план развития;
- **действия:** разбор переписки в задачи и обязательства, утренний бриф, бронь календаря, подготовка к встрече, память о людях и компаниях.

Не SaaS и не облако: всё работает локально, на ваших файлах и подключённых сервисах. Приватность и персонализация — by design.

## Как это работает
1. Склонируйте репозиторий (`git clone`).
2. Запустите `aist-pos-setup` — короткое интервью и установка базы (корневой файл, конвенции, git, хранилища графа).
3. Запустите `aist-pos-assess` — оцените текущий уровень ИИ-компетенций и получите индивидуальный план развития.
4. Используйте навыки-действия по этому плану.

## Установка
- **Claude Cowork:** подключите папки из `skills/` как навыки; скопируйте `config/aist-pos.config.example.yaml` → `aist-pos.config.yaml` (или доверьте это `aist-pos-setup`); подключите нужные сервисы по `config/connectors.md`.
- **Claude Code:** положите `skills/*` в директорию навыков; конфигурация — `config/aist-pos.config.yaml`.

## Навыки
| Навык | Что делает |
|---|---|
| aist-pos-setup | Установка: интервью + база (9 основ) |
| aist-pos-assess | Оценка зрелости → профиль, уровень и план развития |
| aist-pos-entities | Граф: компания (основа) ← люди, проекты |
| aist-pos-extract-tasks | Переписка и транскрипты → задачи и обязательства |
| aist-pos-brief | Утренний бриф (календарь + почта + обязательства) |
| aist-pos-schedule | Бронь календаря (с подтверждением) |
| aist-pos-meeting-prep | Подготовка к встрече (кто / контекст / повестка) |
