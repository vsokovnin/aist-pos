# AIST POS

**Личная операционная система для руководителя — поверх Claude.**

Оцени зрелость своей AI-настройки и расти по шагам: от «болтаю с ботом» до системы, которая помнит контекст, разбирает входящие в задачи, готовит к встречам и ведёт твой граф людей и организаций.

## Что это
Набор портативных скилов для **Claude Cowork / Claude Code**:
- **оценка зрелости** (`aist-pos-assess`) → профиль + стадия + приоритизированный план роста;
- **действия** (US1–US5): разбор переписки в задачи и долги, утренний бриф, бронь календаря, подготовка к встрече, память о сущностях.

Не SaaS и не облако: всё работает локально, на твоих файлах и подключённых коннекторах. Приватность и персонализация — by design.

## Как это работает
1. `git clone` этот репозиторий.
2. Запусти `aist-pos-setup` (/setup) — интервью + установка базы (корневой файл, конвенции, git, граф-сторы).
3. Запусти `aist-pos-assess` — узнай свою стадию и план роста.
4. Используй action-скилы по плану.

## Установка
- **Claude Cowork:** подключи папки из `skills/` как скилы; скопируй `config/aist-pos.config.example.yaml` → `aist-pos.config.yaml` (или дай это сделать `aist-pos-setup`); подключи нужные Connectors по `config/connectors.md`.
- **Claude Code:** положи `skills/*` в директорию скилов; конфиг — `config/aist-pos.config.yaml`.

## Скилы
| Скил | Что делает |
|---|---|
| aist-pos-setup | Установка: интервью + база (9 Must) |
| aist-pos-assess | Оценка зрелости → профиль + стадия + план |
| aist-pos-entities | Граф: организация (база) ← люди, проекты |
| aist-pos-extract-tasks | Переписка+транскрипты → задачи и долги |
| aist-pos-brief | Утренний бриф (календарь+почта+долги) |
| aist-pos-schedule | Бронь календаря (с подтверждением) |
| aist-pos-meeting-prep | Подготовка к встрече (кто/контекст/повестка) |
