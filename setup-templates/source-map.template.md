# Карта источников контекста × чувствительность

| Источник | Где лежит | Уровень доступа / чувствительность |
|---|---|---|
| (пример) Рабочая почта | gmail / outlook | личное, не шарить |
| (пример) Заметки встреч | runtime/, транскрипты | внутреннее |
| (заполни свои) | | |

Сверяться перед подключением/обработкой источника. Не смешивать чувствительные контексты.
