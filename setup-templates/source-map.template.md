# Где что лежит и что конфиденциально

| Материалы | Где лежат | Конфиденциальность |
|---|---|---|
| (пример) Рабочая почта | gmail / outlook | личное, не показывать другим |
| (пример) Заметки встреч | runtime/, транскрипты | внутреннее |
| (заполни свои) | | |

Сверяться, прежде чем дать агенту доступ к новому месту. Конфиденциальное не смешивать с остальным.
