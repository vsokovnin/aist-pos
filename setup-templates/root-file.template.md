## Куда что класть
- Сущности (организации/люди/проекты) → `entities/{org,person,project}/` (схема: `config/entities-schema.md`).
- Задачи и долги → `runtime/tasks.md`. Бриф → `runtime/brief.md`.
- Источники контекста и их чувствительность → `source-map.md`.
- Имена файлов → по `naming-convention.md`.

## Guardrails (от обратного)
