## Как агенту работать (вводные под задачу)
- Перед значимой задачей подними нужные вводные: карточки людей/организаций из `entities/`, открытые задачи/обязательства из `runtime/tasks.md`.
- Стиль: конкретно, без воды.
- Чего не делать: не выдумывать факты о людях/обязательствах; не писать вовне (почта/календарь) без подтверждения.

## Куда что класть
- Карточки базы контактов (организации/люди/проекты) → `entities/{org,person,project}/` (схема: `config/entities-schema.md`).
- Задачи и обязательства → `runtime/tasks.md`. Бриф → `runtime/brief.md`.
- Где что лежит и что конфиденциально → `source-map.md`.
- Имена файлов → по `naming-convention.md`.

## Чего не делать
- Необратимое / вовне (запись в календарь, отправка) → сначала подтверждение.
- Конфиденциальное (по `source-map.md`) → не смешивать с остальным.

## Навыки AIST POS
- `aist-pos-setup` — установка/донастройка.
- `aist-pos-assess` — оценка зрелости → план роста.
- `aist-pos-extract-tasks` · `aist-pos-brief` · `aist-pos-schedule` · `aist-pos-meeting-prep` · `aist-pos-entities` — действия.
